1. sim_data_generation.py.
This file is used to generate the synthetic data.

2. sim_SJDTM.py
This file is the main program of SJDTM for estimating relevant parameters based on the synthetic data generated from sim_data_generation.py.

3. sim_evalution_matrics.py
This file is used to compute the evalution_matrics to measure the performance of SJDTM.

4. DTMpart.py
This file is used for updating topic-word distributions in sim_SJDTM.py.

Users can put the above files in one folder and then directly run sim_SJDTM.py.
